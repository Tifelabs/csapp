        Using only bit-level and logical operations. 
        Write a C expression that is equivalent to \texttt{x === y}
        In other words,It will return 1 when x and y are equal and 0
        otherwise.\\
        \texttt{Solution}\\
        !(x $\oplus$ y)
    \end{flushleft}
