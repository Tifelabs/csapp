    \begin{flushleft}
        Explain how \\
        
        \[
        T2U_w(x) = \begin{cases} x & \text{if } x \geq 0 \\ x + 2^w & \text{if } x < 0 \end{cases}
        \] 
        applies in table Practice Problem 2.19
        \smallbreak
        \textbf{\small Answer}
            This is because the first four entries, the values of $x$
            are negative and $T2U_4(x) = x + 2^4$. For the remaining two entries 
            the values are nonnegative and $T2U_4(x) = x$
    \end{flushleft}
